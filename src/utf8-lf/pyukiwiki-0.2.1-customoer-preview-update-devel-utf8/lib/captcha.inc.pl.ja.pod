=head1 NAME

captcha.inc.pl - PyukiWiki プラグイン

=head1 重要度

★★★☆☆

=head1 概要

SPAM防止の為の画像文字認証を処理する。

=head1 説明

SPAM防止の為の画像文字認証を処理する。

=head1 使用方法

GD モジュールをインストールして下さい。

captcha.inc.cgi にリネームして下さい。

skinディレクトリに、True Type フォントを指定して下さい。

info/setup.cgi に 使用するフォントのパスを $captcha::font 変数に指定して下さい。
=head1 上書きする関数

spam_filter
